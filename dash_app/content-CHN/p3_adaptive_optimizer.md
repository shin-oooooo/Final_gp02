### 图3.1 AdaptiveOptimizer

最优权重向量记为 $w^{\ast}$，由 AdaptiveOptimizer 求解，目标函数随防御等级自动切换：

**目标**：

- **μ_p**：投资组合日收益均值
- **σ_p**：投资组合日波动率

#### Level 0 — 常规态：最大化夏普比率

```
minimize  −(μ_p − r_f) / σ_p     其中 r_f = 0
约束：Σwᵢ = 1，  0 ≤ wᵢ ≤ 1
```

纯数学最优：在给定收益与风险下，找到单位风险回报最高的权重组合。

#### Level 1 — 警戒态：夏普 + 语义惩罚

```
minimize  σ_p  +  λ × Σ(wᵢ × neg_sentiment_i)
```

在控制风险的同时，对情绪分为负的资产加征"分配成本"——情绪越差的资产，其权重越被优化器压低。λ 滑块控制惩罚力度。

#### Level 2 — 熔断态：最小化 CVaR（条件风险价值）

```
R_cond = subset_returns_for_cvar(hist_returns)
minimize  CVaR_α(w)  =  −E[组合收益 | 落在最差 α% 尾部]
```

完全放弃追求收益，目标只有一个：在极端情境（尾部）里少亏损。使用的历史样本也特意只选与当前市场状态（高波动、高相关）相似的历史时期，使 CVaR 计算更贴合实际风险。

##### 此阶段流程图

```
各资产日收益数据（训练集）
        │
        ▼
  μ向量 + 协方差矩阵
        │
       ▼
  AdaptiveOptimizer
  ┌─ Level 0: Max Sharpe → w*
  ├─ Level 1: Min (σ + λ·语义惩罚) → w*
  └─ Level 2: Min CVaR（条件样本）→ w*
        │
        ▼
  μ_p = w* @ μ  （投资组合日收益）
  σ_p = √(w*@Σ@w*)（投资组合日波动）
```
